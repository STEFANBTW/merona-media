# Merona Media - A Cinematic Digital Experience

Merona Media is a high-fidelity, immersive landing page built for the modern age. Rejecting the barren aesthetics of traditional minimalism, this project explores a "Maximalist Modernist" approach, blending Bauhaus-inspired structural discipline with organic, earth-toned richness.

## ✨ Features

- **Cinematic Depth**: Extreme Z-axis typography and ScrollTrigger-driven depth shifts.
- **GSAP Orchestration**: Smooth, frame-perfect animations including an interactive "Lens Showcase" and magnetic UI elements.
- **Responsive Architecture**: Fully responsive design that maintains its cinematic impact across all viewports.
- **Organic × Bauhaus Aesthetic**: A carefully curated color system of Umber, Gold, and Navy, paired with geometric rigor.
- **Interactive Lens**: A forensic examination UI that reveals the detail behind the design artifact.

## 🛠️ Built With

- [Next.js 15+](https://nextjs.org/) - The React framework for the web.
- [Tailwind CSS](https://tailwindcss.com/) - For sophisticated, utility-first styling.
- [GSAP (GreenSock)](https://greensock.com/gsap/) - For high-performance, cinematic animations.
- [Lucide React](https://lucide.dev/) - Clean, geometric icon system.
- [Framer Motion](https://www.framer.com/motion/) - For complementary micro-interactions.

## 🎨 Design Philosophy

> "Abundance is the only truth."

This project follows a strict aesthetic specification:
- **Earth-Toned Palette**: Sand, Umber, Gold, and Clay.
- **Modernist Typography**: Clean geometric sans-serifs paired with humanist serifs.
- **Architectural Grid**: 12-column Bauhaus-inspired layouts.
- **Tactile Texture**: Subtle noise and natural material overlays for a human feel.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
